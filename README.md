# ssl-zhiboba-pages

一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或具体网站，仅作为静态页面集合的存储与展示空间。

## 项目简介

本仓库主要存放独立的 HTML 文件，涵盖各类信息展示页面。所有页面均为静态内容，无需后端支持，可直接通过浏览器访问。

## 目录说明

```
/
├── index.html          # 入口页面（可选）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

- 根目录下的 `index.html` 可作为导航页，列出所归档的页面链接。
- 各页面可存放于独立子目录中，便于管理和引用。

## 页面归档说明

- 所有页面均为纯 HTML 格式，不依赖外部数据库或服务器端脚本。
- 页面内容可包含文本、图片、样式、脚本等常见前端资源。
- 本仓库不保证页面内容的时效性或准确性，仅为归档用途。
